# Responsive Layout Fix

A very simple project where a desktop-only page is converted into a mobile-friendly layout using CSS media queries.

## What Was Done
- Identified fixed-width sections and large images
- Added media queries for screens under 768px
- Stacked columns vertically for mobile
- Reduced text sizes for small screens
- Made navigation stack vertically
- Fixed overflow and scrolling issues
- Ensured images scale properly on all devices

## Tech Used
- HTML5  
- CSS3  
- Chrome DevTools
